# GKJT Redesign (Next.js + MySQL)

This is the enhanced version of the GKJ Tangerang website, featuring a modern design based on Figma.

## Setup
1. Copy `.env.local` if not present.
2. `npm install`
3. `npm run dev`

## Backend
Uses the same MySQL database `gkjtangerang_2024` and API routes as the previous version.

## Design
- Font: Inter / Plus Jakarta Sans (Modern Sans-serif)
- Colors: Primary (Blue/Navy), Accent (Yellow/Gold), Neutral (White/Gray)
